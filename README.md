# Asistente de Python para Examen

Asistente en consola para practicar el examen de programación en Python.
Usa un archivo `python_context.md` con los códigos y apuntes del profesor
como referencia de estilo.

## Requisitos

- Python 3.10+ recomendado
- [Ollama](https://ollama.com/) instalado y corriendo
- Modelo de código (por defecto `qwen2.5-coder:1.5b`)

## Instalación rápida

```bash
pip install -r requirements.txt
